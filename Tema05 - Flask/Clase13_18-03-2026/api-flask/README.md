# Api Flask

## Instalación

### Entorno virtual

`Linux / MacOS`
```bash
python3 -m venv entorno_flask
source entorno_flask/bin/activate
```

`Windows`
```bash
python -m venv entorno_flask
entorno_flask\Scripts\activate
source entorno_flask/Scripts/activate
```

### Dependencias

```bash
pip install Flask
```
